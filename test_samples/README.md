## How to Use

1. **Access the UI**: http://localhost:3002/
2. **Navigate to Step 1**: "Upload Documents (Optional)"
3. **Drag and drop** any of these files or click to browse
4. **Supported formats**: PDF, DOCX, CSV
5. **Continue** to model configuration
